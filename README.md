# BaraageGame
## 說明：<br>
遊戲可使用.JAR檔案開啟，程式為Java語言編寫，可使用Eclipse將檔案匯入查看程式碼，此為大學程式設計課程期末專題因個人興趣製作的小遊戲，旨為熟悉各類別應用、繼承關係、畫布畫筆的使用及多線程等，圖形繪製全程使用Graphics而無匯入外部圖檔。
## 操作：<br>
使用鍵盤方向鍵操作上下左右，Z鍵射擊及選單確認X鍵防護罩，Shift鍵減速移動並集中射擊，遊戲主選單內設有Info可見操作。<br>
